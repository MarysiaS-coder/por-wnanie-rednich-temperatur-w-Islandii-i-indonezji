# porownanie srednich temperatur w Islandii i Indonezji
Projekt poświęcony analizie porównawczej zmian temperatury w dwóch skrajnie różnych strefach klimatycznych. Głównym celem pracy było zbadanie dynamiki zmian pogodowych oraz dopasowanie odpowiednich modeli statystycznych.

# zakres analizy:
* Eksploracyjna analiza danych: porównanie średnich temperatur, wizualizacja przebiegów czasowych.
* Badanie stacjonarności: Wykorzystanie testów statystycznych (np. test ADF) do sprawdzenia, czy parametry statystyczne szeregu są stałe w czasie.
* Identyfikacja składnika losowego
* Analiza trendu: Identyfikacja długookresowych tendencji wzrostowych lub spadkowych w obu lokalizacjach.
* Modelowanie: Dopasowanie modeli (np. ARIMA, ARMA, Holta-Wintersa) w celu opisania struktury danych.
