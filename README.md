# Porównanie średnich temperatur w Islandii i Indonezji - projekt grupowy
Projekt poświęcony analizie porównawczej zmian temperatury w dwóch skrajnie różnych strefach klimatycznych. Głównym celem pracy było zbadanie szeregów czasowych zmian pogodowych oraz dopasowanie odpowiednich modeli statystycznych do danych.

# Zakres analizy:
* Eksploracyjna analiza danych: porównanie średnich temperatur, wizualizacja przebiegów czasowych.
Kompleksowa analiza szeregów czasowych:
* Badanie stacjonarności: Wykorzystanie testów statystycznych (np. test ADF)
* Badanie homoskedastyczności (np. test Breucha-Pagana, White'a, Kruskala-Wallisa, Haririsona-McCabe'a)
* Identyfikacja składnika losowego
* Analiza trendu: Identyfikacja długookresowych tendencji wzrostowych lub spadkowych w obu lokalizacjach.
* Modelowanie: Dopasowanie modeli (np. ARIMA, ARMA, Holta-Wintersa, modele ARCH) w celu opisania struktury danych.
